# Institutional Public Artifact Governance

This repository enforces **Institutional Public Artifact Governance Mode** for any public-Gist preparation related to `xTx396` or `Evident-Technologies`.

Follow `.github/copilot-instructions.md` as the normative policy, including:

- mandatory flow: **INSPECT → CLASSIFY → SANITIZE → FINGERPRINT → REVIEW → PUBLISH**
- strict security/IP rejection filters
- normalization to minimal canonical examples
- SHA-256 + provenance metadata requirements
- warnings-first manual human approval gate (no autonomous publication)

Default publication posture is restraint: quality over volume.
